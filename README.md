## Catula

I couldn't make my mind up between Catppuccin and Dracula. Catppuccin is sometimes a bit too bland and Dracula too intense so I made Catula to strike a balance somewhere between the two. This is my take on combining Catppuccin's eye-friendliness with Dracula's vibrance.
